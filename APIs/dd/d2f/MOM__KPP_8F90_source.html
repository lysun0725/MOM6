1010 !!
1011 !! \section section_KPP The K-Profile Parameterization
1012 !!
1013 !! The K-Profile Parameterization (KPP) of Large et al., 1994, (http://dx.doi.org/10.1029/94RG01872) is
1014 !! implemented via the Community Vertical Mixing package, [CVmix](https://code.google.com/p/cvmix),
1015 !! which is called directly by this module.
1016 !!
1017 !! The formulation and implementation of KPP is described in great detail in the
1018 !! [CVMix manual](https://cvmix.googlecode.com/svn/trunk/manual/cvmix.pdf) (written by our own Stephen Griffies).
1019 !!
1020 !! \subsection section_KPP_nutshell KPP in a nutshell
1021 !!
1022 !! Large et al., 1994, decompose the parameterized boundary layer turbulent flux of a scalar, \f$ s \f$, as
1023 !! \f[ \overline{w^\prime s^\prime} = -K \partial_z s + K \gamma_s(\sigma), \f]
1024 !! where \f$ \sigma = -z/h \f$ is a non-dimensional coordinate within the boundary layer of depth \f$ h \f$.
1025 !! \f$ K \f$ is the eddy diffusivity and is a function of position within the boundary layer as well as a
1026 !! function of the surface forcing:
1027 !! \f[ K = h w_s(\sigma) G(\sigma) . \f]
1028 !! Here, \f$ w_s \f$ is the vertical velocity scale of the boundary layer turbulence and \f$ G(\sigma) \f$ is
1029 !! a "shape function" which is described later.
1030 !! The last term is the "non-local transport" which involves a function \f$ \gamma_s(\sigma) \f$ that is matched
1031 !! to the forcing but is not actually needed in the final implementation.
1032 !! Instead, the entire non-local transport term can be equivalently written
1033 !! \f[ K \gamma_s(\sigma) = C_s G(\sigma) Q_s \f]
1034 !! where \f$ Q_s \f$ is the surface flux of \f$ s \f$ and \f$ C_s \f$ is a constant.
1035 !! The vertical structure of the redistribution (non-local) term is solely due to the shape function, \f$ G(\sigma) \f$.
1036 !! In our implementation of KPP, we allow the shape functions used for \f$ K \f$ and for the non-local transport
1037 !! to be chosen independently.
1038 !!
1039 !! [google_thread_NLT]: https://groups.google.com/forum/#!msg/cvmix-dev/i6rF-eHOtKI/Ti8BeyksrhAJ "Extreme values of non-local transport"
1040 !!
1041 !! The particular shape function most widely used in the atmospheric community is
1042 !! \f[ G(\sigma) = \sigma (1-\sigma)^2 \f]
1043 !! which satisfies the boundary conditions
1044 !! \f$ G(0) = 0 \f$,
1045 !! \f$ G(1) = 0 \f$,
1046 !! \f$ G^\prime(0) = 1 \f$, and
1047 !! \f$ G^\prime(1) = 0 \f$.
1048 !! Large et al, 1994, alter the function so as to match interior diffusivities but we have found that this leads
1049 !! to inconsistencies within the formulation (see google groups thread [Extreme values of non-local transport][google_thread_NLT]).
1050 !! Instead, we use either the above form, or even simpler forms that use alternative upper boundary conditions.
1051 !!
1052 !! The KPP boundary layer depth is a function of the bulk Richardson number, Rib.
1053 !! But to compute Rib, we need the boundary layer depth. To address this circular
1054 !! logic, we compute Rib for each vertical cell in a column, assuming the BL depth
1055 !! equals to the depth of the given grid cell. Once we have a vertical array of Rib(k),
1056 !! we then call the OBLdepth routine from CVMix to compute the actual
1057 !! OBLdepth. We optionally then "correct" the OBLdepth by cycling through once more,
1058 !! this time knowing the OBLdepth from the first pass. This "correction" step is not
1059 !! used by NCAR. It has been found in idealized MOM6 tests to not be necessary.
1060 !!
1061 !! \sa
1062 !! kpp_calculate(), kpp_applynonlocaltransport()
1063 end module mom_kpp
